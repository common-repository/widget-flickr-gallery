Display Flickr photos (by Flickr screen name) in your sidebar. 

== Description ==

This plugin is a widget that can display your latest or favories Flickr photos. 

The features:
* Easy to setup and configure with Screen name
* display in interactive accordion or by small squarre

== Installation ==

1. Upload archive contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Check out your sidebar widgets configuration

If you have a problem when you update the plugin, remove your widget and re-install.
